## Aplicativo de cartão de visits. :bust_in_silhouette:

Criando um App de Cartão de Visitas em Kotlin desenvolvido durante o Bootcamp da Dio. Foram usadas as seguintes tecnologias: 

- Clean Architecture
- MVVM
- Koin
- Room 
- Fragments
- Navigation
